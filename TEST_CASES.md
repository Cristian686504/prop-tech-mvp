## Convenciones

| Símbolo / Valor | Significado |
|------------------|-------------|
| **TC-XXX**       | Identificador único del caso de prueba |
| **Sin ejecutar** | El caso aún no ha sido ejecutado |
| **Pasó**         | El caso fue ejecutado y el resultado obtenido coincide con el esperado |
| **Falló**        | El caso fue ejecutado y el resultado obtenido NO coincide con el esperado |
| **Crítico**      | Bloquea funcionalidad core del sistema |
| **Alto**         | Afecta funcionalidad importante pero no bloquea completamente |
| **Medio**        | Afecta funcionalidad secundaria |
| **Bajo**         | Cosmético o de mínimo impacto |
